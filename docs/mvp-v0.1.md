# Aangan v0.1 MVP

## Source Inputs

Checked on May 17, 2026:

- Pie App Store listing: https://apps.apple.com/us/app/pie-free-events-cool-people/id1509667820
- Pie App Store screenshots via App Store media assets
- Shared cofounder alignment chat: market research on India social/event apps, trust gaps, host reputation, attendance confidence, and no-WhatsApp communication

## Product Frame

Aangan is not primarily an event listing app.

It is:

- Trust infrastructure for offline social plans.
- A creator/host operating system for small IRL gatherings.
- A social-confidence layer that helps users decide whether leaving home is worth it.

## Launch Wedge

Delhi NCR only.

Initial event types:

- Hauz Khas board games and cafe tables.
- Lodhi/Khan Market creator walks.
- Gurgaon founder/operator dinners.
- Curated small groups with visible attendee context.

## Core User Fear

Users are not only asking:

- What is happening?

They are asking:

- Will anyone actually show up?
- Is this safe and non-creepy?
- Are these my kind of people?
- Is the host serious?
- Will I feel awkward if I go solo?
- Will this waste my travel time?

## Current V0.1 Flows

### 1. Fake Login

- Prefilled name, phone, Delhi NCR city, Instagram, LinkedIn, and social intent.
- Starts the prototype with identity and trust proof instead of anonymous browsing.

### 2. Discover

- Dark Pie-like feed.
- All/friends toggle.
- Delhi NCR only.
- Filters: all, free, today, curated, creator-led, friends going.
- Event cards show price, confidence score, organizer, attendee strip, and confirmation count.

### 3. Event Detail

Tap any event to inspect:

- Event description.
- Trust signals.
- Pre-chat prompts.
- Organizer profile, rating, repeat rate, and linked identity.
- Who is going, with role, mutual context, and approval/confirmation status.

### 4. Host

Host creation emphasizes accountability:

- Plan format.
- Venue/locality/time/capacity.
- Public, curated, or private visibility.
- Guest list control.
- Venue proof and pre-event communication checklist.
- Host review impacts future creator rank.

### 5. Chat

Fake in-app chat proves the "no WhatsApp migration" thesis:

- Event pre-chats.
- Host reassurance.
- Route/table proof.
- Prompt-led warmups.
- "Stars aligned" people prompt.

### 6. Profile

Profile is a social portfolio:

- Linked identity.
- Guest rating.
- Plans attended.
- Trust graph mock.
- Comfort settings.

## Differentiators To Test

- Host reputation graph.
- Attendance confidence score.
- Visible guest list before RSVP.
- Structured pre-event socialization.
- In-app communication instead of WhatsApp.
- Curated guest selection by host.
- Identity-driven discovery through Instagram/LinkedIn-style social proof.

## Non-Goals

- Payments.
- Ticketing.
- Full notifications.
- Real chat backend.
- Multi-city discovery.
- AI matching.
- Production moderation.

## Metrics For Real Test

- Save or join rate from Discover.
- Event detail open rate.
- Join request completion.
- Pre-chat message rate.
- RSVP to attended conversion.
- Repeat attendance by event type.
- Host repeat creation.
- Report/block/no-show rate.
